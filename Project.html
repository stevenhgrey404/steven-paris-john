Project εταιρίας


Οι προγραμματιστές της εταιρίας μας έχουν ασχοληθεί με κάποια πρότζεκτ που τους έχουν ανατεθεί όπως για παράδειγμα να δημιουργήσουν ένα πρόγραμμα για δανεισμό βιβλίων από τη βιβλιοθήκη, ένα πρόγραμμα για επιπλοποιείο για να πουλάει έπιπλα και διάφορα μικρά πρότζεκτ με τις γλώσσες προγραμματισμού Java/C++/Python.
Μελλοντικά η εταιρία μας θα ασχοληθεί με την δημιουργία στατικών ιστοσελίδων στα πλαίσια ενός μαθήματος.
