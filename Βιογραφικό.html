Πάρης Ζωηρός


Λίγα λόγια :

-Ο Πάρης Ζωηρός έχει κατάγωγη από το Σιδηρόκαστρο Σερρών και σπουδαζει στη σχολή Μηχανικών Πληροφορικής της Καβάλας.

Βιογραφικό :

-Κατέχει βασικές γνώσεις προγραματισμού και ιδιαίτερα σε Java, Java FX, C++, Python, HTML, Oracle SQL.Επίσης έχει εργαστεί σε διάφορα project εφαρμογών αποκτόντας εμπειρία.

Χόμπυ :

-Εκτός απο τις υποχρεώσεις του ως φοιτητής στον ελεύθερο του χρόνο αθλήται σε ομάδα καλαθοσφαίρησης και γυμνάζεται .Επιπλέον έχει κλήση στη ζωγραφική και στις προσωπογραφιές.Παρα το βάρος των εργασιών της σχολης μελετά επιπλέον βιβλία προγραματισμού για να διευρύνει τις γνώσεις του.
