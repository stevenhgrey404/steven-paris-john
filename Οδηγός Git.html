Οδηγός Git


Για να κάνετε Push ένα αρχείο και να το ανεβάσετε στο Github πρέπει να πατήσετε το κουμπί Commit για την επιβεβαίωση του ανεβάσματος.
Αν θέλετε να κατεβάσετε αρχεία πρέπει να τα κάνετε Pull έτσι ώστε να έχετε την τελευταία έκδοση.
